Project Krypton

Krypton is an end-to-end encrypted, cloud-based open source messenger, programmed as part of a Matura thesis. Even though the software is currently in the beta phase, the essential features have been implemented.
It is made of a server and client-side application. Since the server is run in a docker container, it can be run on any environment capable of running docker, as testing environment a server has been configured under https://kr.ttschnz.ch/, but it is unclear how long this will be available to the public (as of end of 2021: online until summer 2022).
